What is this keyword in JS?

This keyword points to the current object ie. most of the time it points to global object which is window object.

            I/p: 

            var obj={
                name: "vivek",
                age:22,

                printName:function(){
                    console.log("name is "+this.name+" and age is "+this.age)
                }
            }

            O/p: 
            name is vivek and age is 22

What is call,apply,bind in JS?

Call apply bind is basically used to borrow function methods.

Call:It borrows the function method and passes the object as an argument

            I/p: 
            var obj={
                name: "rahul",
                age:34,
                
            }

            function a(degree){
                console.log("name is "+this.name+" age is "+this.age+" and degree is "+degree);
            }

            a.call(obj,"b.tech");

            O/p: 
            name is rahul age is 34 and degree is b.tech

Apply:It borrows the function method and passes the object as an argument.The only difference is it passes additional argument inside the array.

            I/p: 
            var obj={
                name: "rahul",
                age:34,
                
            }

            function a(degree){
                console.log("name is "+this.name+" age is "+this.age+" and degree is "+degree);
            }

            a.apply(obj,["b.tech"]);

            O/p: 
            name is rahul age is 34 and degree is b.tech

Bind:It borrows the function method and passes the object as an argument and returns a new function ie. copy of that function.

        I/p: 
        var obj={
            name: "rahul",
            age:34,
            
        }
        
        function a(degree){
            console.log("name is "+this.name+" age is "+this.age+" and degree is "+degree);
        }
        
        var tempBind=a.bind(obj,"b.tech");
        tempBind();

        O/p: 
        name is rahul age is 34 and degree is b.tech

What is prototype chain in JS?

Each object has a private property which holds a link to another object called its prototype.That prototype object has a prototype has a prototype of its own and so on until it reaches to null as its prototype.Hence this chain is called prototype chaining.

            I/p: 
            var obj={
                name: "vivek",
                age:22
            }

            console.log(obj.__proto__.proto__);


            O/p:
            null
