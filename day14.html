What is callback and callback hell?

callback

Callback function is a function which can be passed as an argument to another function.It can be also executed later.

          I/p: 
          setTimeout(()=>{
              console.log("settimeout called").;
          },2000)

          O/p: 

          console.log("settimeout");

Callback hell

Callback hell is a phenomenon which happens when multiple callback is called which causes a deadlock situation due to which it becomes difficult to debug the code.

         setTimeout(()=>{
             console.log("callback 1");

             setTimeout(()=>{
                 console.log("callback 2");

                 setTimeout(()=>{
                     console.log("callback 3");
                 },3000)
             },2000)
         },1000)

What are promises?

Promises are javascript objects which tells whether a asynchronous task gets resolved or rejected.It is used to prevent callback hell. 


            I/p:

            var pObj=new Promise((resolve,reject)=>{
                var a=2;
                if(a==2)
                resolve();
                else
                reject();
            })

            pObj.then(()=>{
                console.log("task successful").catch(()=>{
                    console.log("task failed");
                })
            })


            O/p: 

            task successful
