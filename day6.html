What is Function?

Function is a block of code which is excuted for a particular task.Here we write the code once and can use it multiple times.

What is Function declaration?

Function declaration defines a function with specified parameters.Here we have to give function name.Function declarations are hoisted.

             Ex:

             function add(a,b);
             {
                 return a+b;
             }

             console.log(add(2,3));

What is function Expression

Function Expression is similar to function declaration but here the function name is ommited.Function expression are not hoisted since during hoisting it treats it as a function variable.

             Ex: 
             const add=function(a,b){
                 return a+b;

             }

             console.log(add(2,3));

What is a constructor function?

In Javascript the constructor function is used to create objects.We use cosntructor function by using new keyword

             Ex:

             const add=new Function("a","b","return a+b");
              console.log(add(2,3));

What is IIFE?

IIFE is immediately invoked function expression which gets exceuted as soon as it is created.It is basically a anonymous function which can be executed only once.

             (function(){
                 console.log("this is a anonymous function");
             })();

What is a first class function?

First class function is function which can be passed as an argument inside a another function or it can be returned by a function or it can assigned as a value to another function.

             Ex:
             function sum(a,b){
                return a+b;
            }
            
            function sumCall(sum){
            
               return sum(2,3);
            }
            
            console.log(sumCall(sum));

            //Here sum is a first class function

What is a callback function?

Callback function is a function which can be passed as an argument to another function.

             Ex:

             function sum(a,b)
             {
              console.log("sum"+a+b);
             }

             function sumCall(sum){
                 sum(2,3);
             }

             sumCall(sum);

             // Here sum is a callback function;

What is a higher order function?

Higher order function is a function which can accept a function as a parameter or return a function.

             Ex:

             function sum(a,b)
             {
              console.log("sum"+a+b);
             }

             function sumCall(sum){
                 sum(2,3);
             }

             sumCall(sum);

             // Here sumCall is a higher order function

What are closures?

Any function maintaining lexical scoping is called as closures.Here the function will try to access the variables in local scope,if they dont find it then it will search it in the outer scope and still if it fails to find it will search in global scope and finally if it is not found then it will return undefined.

            Ex:
            var x=2;
            function a(){
                console.log(x);
