What is a Map in JS?

Map is array method which is used to transform an array.It calls the a callback function for every element and iterate through the array from beginning to end. 

            I/p:
            var arr=[1,2,3,4];
            arr.map((currentVal,index,arr)=>{
                console.log(currentVal+" ");
            })

            O/p:
            1 2 3 4

What is a filter?

Filter is a array method which is used to filter out the partcular data from the array according to the given condition and returns a new array.


            I/p:

            var arr=[1,2,3,4];
            var data=arr.filter((currentVal)=>currentVal>1);
            console.log(data);

            O/p:
            2 3 4

What is a reduce?

Reduce is a array method which calls a callback function for each element and finally returns a single value ie. accumulated value.

            I/p:
            var arr=[1,2,3,4];
            var result=arr.reduce((total,currentVal,index)=>{
                total+=currentVal;
                return total;
            },0)
            console.log(result);

            O/p:
            10

What is Some method in JS?

Some is a array method which returns true if single element satisfies the condition else returns false.

         I/p:
         var arr=[1,2,3,4];
         var data=arr.some((val)=>val>1)
         console.log(data);

         O/p:
         true

What is Every method in JS?

Every is s array method which returns true if every element satisfies the condition else returns false.

         I/p:

         var arr=[1,2,3,4];
         var data=arr.every((val)=>val>1)
         console.log(data);

         O/p:
         false

Slice is a array method which returns selected elements from the array as per the starting index and end index.
