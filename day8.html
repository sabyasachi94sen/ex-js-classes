Length in a string tells the length of the string ie. how many characters are present in the string

            I/p:
            var str="hello";
            console.log(str.length);

            O/p: 
            5

What is splice in string?

Splice is a method used to add/remove array elements.It also modifies the original original.

            I/p: 

            Add elemnts:
            var arr=["hello","hogwards","harry","potter"];
            arr.splice(3,0,"dumbledore");
            console.log(arr);

            O/p: 
            ['hello', 'hogwards', 'harry', 'dumbledore', 'potter'];

            I/p:

            Remove elements:

            I/p: 
            var arr=["hello","hogwards","harry","potter"];
            arr.splice(0,1);
            console.log(arr);

            O/p: 
            ['hogwards', 'harry', 'potter']
