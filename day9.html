What is a object in JS?

Object is a no primitive datatype which stores information as a key value pair

Ex: var obj={ name: "vivek", age: 24, graduation: "b.tech" }

what are the different ways to create objects in JS?

There are 4 ways to create objects in JS :

  • Object using constructor.
  • Object using object literal.
  • Object using function constructor.
  • Object using Object.create method

Object using constructor

            var obj=new Object();
            obj.name="rahul";
            obj.age="23";
            obj.graduation="bba"

Object using object literal

            var obj={
                name: "vivek",
                age: 24,
                graduation: "b.tech"
            }

Object using function constructor

             function sum(a,b){
                this.a=a;
                this.b=b;
                return a+b;
            }

            var sum1=new sum(2,3);
            var sum2=new sum(4,5);

Object using Object.create method: This method allows us to create a new object using existing object as the prototype of the newly created object. 

            var obj={
                name: "ajay",
                age: 45,
                graduation : "bca"
            }

            var newObj=Object.create(obj);

What is object keys and value properties in JS?

Object keys is a object property which returns all the keys of the object.

            I/p: 
            var obj={
                name: "ajay",
                age: 45,
                graduation : "bca"
            }

            console.log(Object.keys(obj));

            O/p: 
            [name,age,graduation]

Object values is a object property which returns all the values of the object.

            I/p: 
            var obj={
                name: "ajay",
                age: 45,
                graduation : "bca"
            }

            console.log(Object.values(obj));

            O/p: 
            [ajay,45,bca]

What is a spread operator?

Spread operator is used to unpack values from array expressions,string or object with more than 0 arguments or elements are expected.

            I/p: 

            var obj={
                name: "deepak",
                age: 22
            }

            var obj1={
                ...obj,
                graduation: bba
            }
            console.log(obj1);

            O/p:

            obj1={
                name: "deepak",
                age: 22,
                graduation: bba
            }

What is this keyword in JS?

This keyword points to the current object ie. most of the time it points to the global object only means the window object.

            I/p: 
            var obj={
                name: "rahul",
                age: 34,
                printDetails: function(){
                    console.log("name="+this.name+" and age"+this.age)
                }
            }

            obj.printDetails();

            O/p: 
            name=rahul and age=34
